## Installation
Not containerized (yet), make sure you have install these locally before running the app:
```bash
brew install python3
brew install freetds
sudo -H pip3 install pymssql
pip3 install flask
```

Run
`python3 app.py`

Point your browser to
`http://localhost:8000`
